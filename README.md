## 安装

Clone the repo and install dependencies:

```bash
npm install
npm start
```


## 打包

To package apps for the local platform:

```bash
npm run package --mac
```

## 生成的mac软件的位置在
```
memory-t-t/release/build/mac/ElectronReact.app

memory-t-t/release/build/mac/记忆胶囊.app // "productName": "记忆胶囊",
```
